# 📊 Superstore Profit Analysis | Power BI

## Overview
This project analyzes Superstore sales and profit data using Power BI to identify business performance across regions, categories, customer segments, and products. The interactive dashboard helps uncover profit trends and supports data-driven decision-making.

## Objectives
- Analyze overall sales, profit, and quantity sold.
- Compare profit across regions and product categories.
- Identify profitable customer segments.
- Track monthly profit trends.
- Highlight loss-making sub-categories.

## Dashboard Highlights
- KPI Cards (Sales, Profit, Quantity)
- Profit by Region
- Profit by Category
- Profit by Customer Segment
- Monthly Profit Trend
- Profit by Sub-Category
- State Filter

## Key Insights
- Technology is the highest-profit category.
- Corporate customers generate more profit.
- South region performs the best.
- Phones are the most profitable sub-category.
- Tables and Binders generate losses.

## Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel
